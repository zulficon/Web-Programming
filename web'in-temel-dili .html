Web'in Temel Dili: HTML

Zülfikar Durak tarafından Cumartesi, 16 Kasım 2024 tarihinde yayınlandı

Tüm modern web siteleri ve web uygulamaları üç temel teknoloji kullanılarak oluşturulmuştur: HTML, CSS ve JavaScript. Bunlar web'in dilleridir.

Bu yazıda HTML'e odaklanalım. HTML'nin ne olduğunu ve sizin de neden öğrenmeniz gerektiğini öğreneceğiz.

HTML Nedir?

HTML, Hyper Text Markup Language'ın kısaltmasıdır. Web geliştiricilerinin bir web sayfasının içeriğini yapılandırmak ve tanımlamak için kullandığı bir işaretleme dilidir (bir programlama dili kesinlikle değildir. karışltırmayınız).

HTML, farklı içerik türlerini tanımlayan öğelerden oluşur: paragraflar, bağlantılar, başlıklar, resimler, videolar, vb. Web tarayıcıları HTML'i anlar ve HTML kodunu web siteleri olarak işler.

HTML'de her bir eleman 3 bölümden oluşur:

  1. Açılış tag
  2. Kapanış tag
  3. Gerçek elementler

Daha fazla bilgi edinmek için şu adresi ziyaret edebilirsiniz: MDN Web Docs.

Neden HTML öğrenmelisiniz peki?

Web'in temel dilini öğrenmek için sayısız neden var. İşte bunlardan 5 tanesi:

  • Temel web geliştirme dilini kullanabilmek
  • Wordpress veya Wix gibi araçlara güvenmek yerine güzel web sitelerini elle tasarlamak
  • Web uygulamaları oluşturmak için
  • Arkadaşlarınızı etkilemek için
  • eğlenmek için ❤️
